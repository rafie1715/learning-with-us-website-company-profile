# Learning With Us — Company Profile (Vite + React)

Simple, responsive company profile scaffold for Learning With Us.

Run locally:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Deploy:
- Use Vercel or Netlify: connect the repo and set build command `npm run build` and publish `dist/`.

Notes:
- Pages included: Home, About, Vision & Mission, Products & Services.
- Edit content in `src/pages` to customize.
